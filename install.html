Installation Notes

Installation Notes

transducers-scala is published to Sonatype and can be installed with
   your favourite dependency manger:

  • sbt
  • leiningen
  • gradle
  • maven

SBT

  libraryDependencies += "de.knutwalker" %% "transducers-scala" % "0.2.1-SNAPSHOT"

Leiningen

  [de.knutwalker/transducers-scala "0.2.1-SNAPSHOT"]

Gradle

  compile 'de.knutwalker:transducers-scala:0.2.1-SNAPSHOT'

Maven

  <dependency>
    <groupId>de.knutwalker</groupId>
    <artifactId>transducers-scala</artifactId>
    <version>0.2.1-SNAPSHOT</version>
  </dependency>

Other Dependencies

transducers-scala has no additional dependencies besides scala 2.11.5.

Other Modules

transducers-scala also comes with the following additional modules:

  • "de.knutwalker" % "transducers-scala-reactivestreams" % "0.2.1-SNAPSHOT"

    • Depends on:
    • "org.reactivestreams" % "reactive-streams" % "0.4.0"

  • "de.knutwalker" % "transducers-scala-rxscala" % "0.2.1-SNAPSHOT"

    • Depends on:
    • "io.reactivex" % "rxscala" % "0.23.1"
